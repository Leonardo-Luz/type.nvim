## type.nvim [WIP]

**Dependencies:** This plugin require `leonardo-luz/floatwindow`

**Installation:**  Add `leonardo-luz/type.nvim` to your Neovim plugin manager (e.g., `init.lua` or `plugins/type.lua`).

**Usage:**
* Type `:Type` to start
    * `<bs>` back one word
    * `<space>` advance one word
